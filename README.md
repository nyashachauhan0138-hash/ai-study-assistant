# AI Study Assistant

An AI-powered document assistant that allows users to query PDFs using Retrieval-Augmented Generation (RAG).

## Features
- Multi-PDF upload
- Semantic search using embeddings
- Context-based answers
- Fallback to general knowledge
- Chat interface

## Tech Stack
- Python
- Streamlit
- Ollama (Llama 3.1)
- ChromaDB
- Sentence Transformers

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
